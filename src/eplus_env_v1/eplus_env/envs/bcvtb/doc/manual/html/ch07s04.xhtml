7.4. Version control

The development site of the BCVTB is at https://github.com/lbl-srg/bcvtb

To obtain an account, email MWetter@lbl.gov.

Please always keep the master branch of the repository in a working condition and work on a github branch for development and testing. Prior to committing changes to the master, make sure that all unit tests work without an error. How to run unit tests is described in Section 7.6, “Compiling the BCVTB”.
